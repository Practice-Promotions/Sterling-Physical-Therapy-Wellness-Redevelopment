== Description ==
The Easy Import Yoast SEO Meta Using CSV makes it easy to bulk import your SEO Meta to the Yoast WordPress SEO plugin in less than 10 minutes. Imoprt support .csv file.

Gives WordPress SEO by Yoast CSV import support so you can update your site's SEO data in bulk.

= Features =
- Bulk Edit Yoast SEO
- Create custom meta tags
- Export Yoast Meta.
- Export Selected Option data post and page data in CSV file

== Frequently Asked Questions ==
= Why you should use the Yoast WordPress Easy Import Yoast SEO Meta Using CSV..? =
- Import new posts, pages, or custom post types and configure their SEO Meta all in the same import.


== Installation ==

Installing the plugin is easy. Just follow these steps:

1. From the dashboard of your site, navigate to Plugins --> Add New.
2. Select the Upload option and hit "Choose File."
3. When the popup appears, select the import-yoast-seo-meta.zip file from your desktop.
4. Follow the on-screen instructions and wait till the upload completes.
5. When it's finished, activate the plugin via the prompt. A message will display confirming activation was successful.

That's it! Just configure your settings as you see fit, and you're on your way to send notification your style.
